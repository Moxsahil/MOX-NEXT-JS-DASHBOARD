# MOX School Management Dashboard

###

<h1 align="start">Hey there 👋</h1>

###

A Beautiful UI dashboard designed in Next.Js, using prisma postgres SQL database and docker's container to stores admin, students, teachers, parent and many more data. A complete full stack application where we can store every small information about each and every thing about school management, expenses. 

###


## Installation
To run this project, Follow the following steps:->

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**
```bash
git clone https://github.com/Moxsahil/MOX-NEXT-JS-DASHBOARD.git
cd MOX-NEXT-JS-DASHBOARD


```
**Download Node Modules**

```bash
  npm install 
```
**Run the Project**  
```bash
  npm run dev 
```
**Run Prisma postgres database**
```bash
  npx prisma studio 
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

** env file setup**  
`DATABASE_URL`

`NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`

`CLERK_SECRET_KEY`

`NEXT_PUBLIC_CLERK_SIGN_IN_URL`

`NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME`

`NEXT_PUBLIC_CLOUDINARY_API_KEY`


**Set-up Docker Conatiner **

Search postgres and then set-up the required username and db password, then the run the docker container



## Tech Stack

**Frontend:** Next.Js, TailwindCSS

**Server:** Node, Docker, Clerk

**Database:** PostgresSQL
